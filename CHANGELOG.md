# Developer Experience Working Group Changelog

This changelog tracks all notable changes, improvements, and updates to the Developer Experience Working Group portal and initiatives. It follows [Keep a Changelog](https://keepachangelog.com/en/1.0.0/) principles.

The format is based on [Semantic Versioning](https://semver.org/), and this project adheres to the versioning scheme: `MAJOR.MINOR.PATCH`.

## Structure

Each version section contains:
- **Added**: New features, sections, or capabilities
- **Changed**: Changes in existing functionality
- **Fixed**: Bug fixes and resolution of issues
- **Removed**: Deprecated or removed features
- **Security**: Security improvements or vulnerability fixes

---

## 2.2.0 - January 2025

### Added

- GitHub Pages deployment workflow with GitHub Actions as publishing source
- Complete homepage redesign with Intersect MBO branding
- Orange gradient "Join Intersect" button with color #f55521
- GitHub-focused community support section
- Developer Experience Initiative feature cards section
- Intersect MBO logo and favicon integration
- Professional button styling with 100% rounded corners
- Dark mode support throughout the portal
- Custom brand color palette (#0128aa blue theme)

### Changed

- Redesigned homepage from 4 sections to 2 focused sections
- Updated navbar with dropdown Resources menu
- Enhanced footer with Intersect ecosystem links
- Improved typography with Inter and JetBrains Mono fonts
- Updated site tagline to reflect community-driven initiatives

### Fixed

- GitHub Pages deployment configuration (GitHub Actions as source)
- Broken links resolved (onBrokenLinks strict validation enabled)
- Navbar hover visibility for brand text and logo
- Dark mode navbar text color (plain white)
- Dark mode footer copyright visibility
- Active sidebar menu items display in blue
- Button and link visibility in dark mode

### Technical Improvements

- Configured `onBrokenLinks: "throw"` for build validation
- Updated all navigation links to point to existing pages
- Enhanced CSS with proper dark mode overrides
- Improved responsive design for mobile devices
- Professional hover effects and transitions
