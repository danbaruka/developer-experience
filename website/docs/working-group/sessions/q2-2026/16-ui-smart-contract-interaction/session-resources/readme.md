# Session 16: UI ↔ Smart Contracts (Wallets + Tx flow) - Resources

Curated references for building UI apps that interact with Cardano validators (Aiken/Plutus).

## Standards & fundamentals

- **CIP-30 (dApp connector / browser wallet API)**: [CIP-0030](https://github.com/cardano-foundation/CIPs/tree/master/CIP-0030)
- **CIPs repository (all specs)**: [cardano-foundation/CIPs](https://github.com/cardano-foundation/CIPs)
- **Reference scripts (CIP-33)**: [CIP-0033](https://github.com/cardano-foundation/CIPs/blob/master/CIP-0033/README.md)
- **Cardano Developer Portal (get started + integrate)**: [developers.cardano.org](https://developers.cardano.org/docs/get-started/)

## Smart contracts (on-chain)

- **Aiken** (recommended track): [aiken-lang.org](https://aiken-lang.org)
- **Plutus docs** (types, datums/redeemers context): [plutus.readthedocs.io](https://plutus.readthedocs.io/en/latest/)

## Off-chain / tx building (SDKs)

Pick one SDK lane and standardize within your project:

- **MeshJS** guides: [meshjs.dev/guides](https://meshjs.dev/guides)
- **Lucid** (ecosystem docs and references): [spacebudz/lucid](https://github.com/spacebudz/lucid) *(check for the currently maintained variant in your org’s stack)*
- **Blaze** (ecosystem references): [butaneprotocol/blaze-cardano](https://github.com/butaneprotocol/blaze-cardano)

## Data providers & infra

- **Blockfrost**: [blockfrost.io](https://blockfrost.io)
- **Koios**: [koios.rest](https://koios.rest)
- **Ogmios** (node bridge + tx evaluation patterns): [ogmios.dev](https://ogmios.dev)
- **Kupo** (UTxO indexer): [cardanosolutions.github.io/kupo](https://cardanosolutions.github.io/kupo/)

## Testing / networks

- **Cardano testnets (Preview/Preprod)**: [docs.cardano.org](https://docs.cardano.org/cardano-testnets/getting-started/)
- **Aiken tests**: [Aiken tests](https://aiken-lang.org/language-tour/tests)
